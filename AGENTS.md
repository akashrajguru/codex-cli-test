# Chrome Deno Game Run clone

This is a nextjs project that is a game like the deno run from the chrome browser.

## Rules

### Nextjs

Use only nextjs client components.

### Tailwind CSS

Use tailwind css for styling.

### Tests

Write test for every feature you write.

### CI

Use Hausky for pre-commit hooks.

#### Run git add . and husky after each feature

After you finish a feature, please run this.

```bash
git add .
.husky/pre-commit
```
